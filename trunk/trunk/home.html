2. By calling an URL :

http://rsstodolist.appspot.com/add?name=somename&url={{ random_url }}
(replace somename by whatever name you want).
Parameter "&title" is optional. If not set, app will get <title> from specified url.

2. Get your feed :

http://rsstodolist.appspot.com/?name=somename
(replace somename by whatever name you want).
