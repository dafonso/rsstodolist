RSS TodoList

1. Add an URL to your feed

1. By using that form :

 : 
 :  replace by whatever name you want

2. By calling an URL :

http://rsstodolist.appspot.com/add?name=somename&url={{ random_url }}

  • By default, the feed title will be fetched from the web page title. You may force it by using the "title" parameter.
  • You may also use short parameter like "n" for "name", "u" for "url" or "t" for "title" like that : http://rsstodolist.appspot.com/add?n=somename&u={{ random_url }}

3. By using a Firefox JetPack extension :

First, install JetPack, then, come back here and click on the install button in the top of that page.

You should then have a new item in your status bar allowing you to type your feed name and use the "add url" button to add the url to the current tab to your feed.

2. Get your feed :

http://rsstodolist.appspot.com/?name=somename
